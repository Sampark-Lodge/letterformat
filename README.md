# LetterFormat.in

Free letter and application formats for Indian users.

## Features

- 30+ ready-to-use letter formats
- Copy-paste functionality
- SEO optimized
- Deployable to GitHub Pages

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Push to GitHub - GitHub Actions will automatically deploy to GitHub Pages.

**Live URL**: https://sampark-lodge.github.io/letterformat/
